# Testes da API

Este projeto utiliza a biblioteca nativa Node.js test para validar as rotas HTTP da API.

## Como executar

1. Instale as dependências do projeto:
   npm install
2. Execute a suíte de testes:
   npm test

Os testes usam mocks de camada de modelo para garantir que a API responda corretamente sem depender de uma instância real do MySQL.
